# Today Date

Este é um projeto simples desenvolvido com HTML, CSS e JavaScript que exibe a data e a hora atual do usuário.

##  Funcionalidades
- Exibe a data atual formatada automaticamente
- Exibe a hora atual do sistema
- Interface simples

## 🛠️ Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript 

##  Objetivo
Este projeto foi criado com o objetivo de praticar manipulação de objetos em JavaScript e trabalhar com datas usando a API `Date`.
